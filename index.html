Elemento Que es? Que haces Ejemplo
<html> </html> Etiqueta Le indica al navegador que es una página web HTML. Se abre arriba de todo y se cierra abajo de todo <html>contenido de la página web </html>
<head></head> Etiqueta Le indica al navegador que acá comienza la información que debe tener sobre la página web. Se abre después de <html> y se cierra al terminar el contenido del head. <head>contenido del head</head>
<title></title> Etiqueta Le indica al navegador el título de la página web (se ve arriba en la barra del navegador).Va dentro del head <title> Mi primera página web</title>
<style type="css"></style> Etiqueta (style type)atributo (text/css) Le indica al navegador el estilo de la página web en HTML. Va dentro del head. Incluye los estilos que tienen los diferentes elementos de la página web >
<body></body> Etiqueta Encierra el contenido de la página web que ve el usuario. <body><h1>Título</h1><p>Contenido que tendrá el párrafo.</p></body>
